About 6-Disc Changer

The 6-Disc Changer is a nostalgic tribute to the CD changers of the 90s and early 2000s, reimagined for the digital streaming age.

What is a 6-Disc Changer?

Back in the day, a 6-disc CD changer was a premium feature in home and car audio systems. It allowed music enthusiasts to load multiple CDs at once and switch between them without the hassle of physically changing discs each time.

Our Vision

While streaming services like Spotify have revolutionized how we access music, they've also changed how we experience albums. The 6-Disc Changer app brings back the intentional, album-focused listening experience that many music lovers miss.

How It Works

With 6-Disc Changer, you can:

  • Select up to 6 albums from your Spotify library
  • Switch between albums like changing discs
  • Enjoy full album experiences rather than shuffled playlists
  • Use familiar controls inspired by classic CD players

Technology

This application is built using:

  • Rust with the Axum web framework
  • MiniJinja for HTML templating
  • HTMX for interactive user experience
  • SQLite database for storage

This project is open source and still under active development.
